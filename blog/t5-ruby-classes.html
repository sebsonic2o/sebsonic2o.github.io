Ruby Classes
Or thinking about Objects...

Like some other programming languages, Ruby is Object-Oriented. This means that everything is represented and manipulated as an Object. What is an Object then? It is basically anything that has its own state and behavior. And we can define our own Objects to best model anything. However, all things (i.e. Objects) are different, but many are kind of the same. This is what a Class is. In Ruby, when defining our own Objects, we will first create a Class to say what kind of Objects we want to create.

For instance, we may think of creating a 2015 Mercedes-Benz C63 AMG coupe. It is kind of a car, isn't it? So let's first create a Class Car in Ruby:

class Car

end

We want to specify how to create our car. This is a special behavior of our car. In Ruby, a behavior of an Object is an instance method. And the special instance method name for creation is initialize:

class Car

  def initialize
  end

end

We can use the year (2015), brand (Mercedes-Benz), model (C63 AMG), and style (coupe) to differentiate it from other cars. All those characterictics for our car should be saved. We should do this at creation. In Ruby, we use instance variables, starting with a @:

class Car

  def initialize(year, brand, model, style)

    @year = year
    @brand = brand
    @model = model
    @style = style

  end

end

Furthermore, we might want to add its state out of the factory. For instance, its mileage:

class Car

  def initialize(year, brand, model, style)

    @year = year
    @brand = brand
    @model = model
    @style = style

    @mileage = 0

  end

end

A car is only good if it can be driven. This will change our mileage. This is another behavior of our car. So we define another instance method to drive:

class Car

  def initialize(year, brand, model, style)

    @year = year
    @brand = brand
    @model = model
    @style = style

    @mileage = 0

  end

  def drive(distance)

    @mileage += distance

  end

end

Now, we can create our 2015 Mercedes-Benz C63 AMG coupe and take it for a first drive:

our_car = Car.new(2015, "Mercedes-Benz", "C63 AMG", "coupe")

our_car.drive(10)
